TASKCODE

Predicting construction worker tasks from GPS data

GPS History

Can a machine learn to tell what someone was working on at a construction site using high-accuracy GPS data?

.

Task Suggestion

After a task is completed, we can algorithmically determine a short list of which kind of task it might have been. A worker would just have to tap the right option.

.

Accurate Predictions

Using N weeks worth of collected data, the top 3 predicted task IDs contain the right task 99% of the time. Removing the difficulty of looking up task IDs and the high chance of errors could dramatically increase the accuracy and reliability of the logging system. Given enough data, it could even be possible to automatically select the right task without user feedback.

.

Complex System

These results were possible with about ten different tasks and dozens of different individual working locations.
